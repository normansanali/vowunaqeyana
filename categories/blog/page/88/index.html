When we should use LinkedHashMap?

LinkedHashMap can be used to maintain insertion order, on which keys are inserted into Map or it can also be used to maintain an access order, on which keys are accessed. This provides LinkedHashMap an edge over HashMap without compromising too much performance. Which is better LinkedHashMap or HashMap? While both HashMap and HashMap classes are almost similar in performance, HashMap requires less memory than a LinkedHashMap because it does not guarantee the iterating order of the map, which makes adding, removing, and finding entries in a HashMap relatively faster than doing the same with a LinkedHashMap.
